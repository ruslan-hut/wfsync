B2B: customer_group ∈ {-1, 6, 7, 16, 18, 19}. -1 is the synthetic flag for direct API callers and for clients promoted from a wFirma-stored NIP.
OSS rate source: order's tax_value derives the rate; if missing, the internal VATProvider (or hardcoded defaultEURates) is used.
Internal > OpenCart: on a B2C EU non-PL mismatch the internal rate replaces the OpenCart-derived rate and a warning is logged.
VIES: when a TaxId is present, the VIES check is non-blocking; failure is logged at WARN with the error topic.
Source of truth: internal/wfirma/invoice.go (flow), internal/wfirma/vat.go (resolveGoodsVatCode), internal/wfirma/vat-codes.go (OSS ID resolution), internal/wfirma/contractor.go (lookup & B2B promotion).
